# Icon Resources

This directory contains icon files for the PS5 Dump Runner FTP Installer.

## Required Icons

- `app_icon.ico` - Application icon (256x256, multi-resolution ICO)
- `connected.png` - Connection status: connected (16x16 or 24x24)
- `disconnected.png` - Connection status: disconnected (16x16 or 24x24)
- `official.png` - Official release indicator (16x16 or 24x24)
- `experimental.png` - Experimental build indicator (16x16 or 24x24)

## Placeholder Status

Icons are placeholders and should be replaced with actual graphics before release.
